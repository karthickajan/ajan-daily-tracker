## ‚ö° Quick Start (3 steps)

### Step 1: Create the Database Table

1. Go to: https://app.supabase.com
2. Select your **Daily Tracker** project
3. Open **SQL Editor** (left sidebar)
4. Click **New Query**
5. Paste this SQL and click **Run**:

```sql
CREATE TABLE tracker_entries (
  id BIGSERIAL PRIMARY KEY,
  device_id TEXT NOT NULL,
  date DATE NOT NULL,
  sleep_6h BOOLEAN DEFAULT FALSE,
  sleep_hours DECIMAL DEFAULT 0,
  bathed BOOLEAN DEFAULT FALSE,
  hair_controlled BOOLEAN DEFAULT FALSE,
  ate_enough BOOLEAN DEFAULT FALSE,
  protein_ok BOOLEAN DEFAULT FALSE,
  dsa_studied BOOLEAN DEFAULT FALSE,
  dsa_hours DECIMAL DEFAULT 0,
  sysdesign_studied BOOLEAN DEFAULT FALSE,
  sysdesign_hours DECIMAL DEFAULT 0,
  deepwork_90min BOOLEAN DEFAULT FALSE,
  solved_designed BOOLEAN DEFAULT FALSE,
  low_distraction BOOLEAN DEFAULT FALSE,
  progress DECIMAL DEFAULT 0,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  UNIQUE(device_id, date)
);

CREATE INDEX idx_device_date ON tracker_entries(device_id, date);
```

### Step 2: Enable Row Level Security (RLS)

1. Go to **Authentication** ‚Üí **Policies**
2. Find `tracker_entries` table
3. Click **New Policy** and select:
   - ‚úÖ Enable read access for all users
4. Repeat and select:
   - ‚úÖ Enable insert access for all users
5. Repeat and select:
   - ‚úÖ Enable update access for all users

### Step 3: Done! Start Using üéâ

```bash
python daily_tracker_web.py
```

Then open: **http://localhost:5000**

## How It Works

1. Browser generates a unique **Device ID** (stored locally)
2. All your entries are linked to that Device ID
3. When you save, data goes to Supabase ‚òÅÔ∏è
4. On any device with same ID, you see all your data
5. If offline, saves locally ‚Üí syncs when back online

## Troubleshooting

**Data not saving?**
- Check browser console (F12 ‚Üí Console tab)
- Verify Supabase project is active
- Make sure `tracker_entries` table exists

**Want to use on another device?**
- Copy the Device ID from your current device
- Paste it in browser console on new device: `localStorage.setItem('ajanDeviceId', 'your_device_id')`
- Refresh page

**Export your data?**
- Click **üì§ Export Data** button to download JSON
